## 🛠️ **Our Tool Stack**

### **Version Control & Code Management**
- **GitHub** - Primary repository hosting and code collaboration
  - URL: [github.com/meetball](https://github.com/meetball)
  - DRI: Engineering Team
  - Purpose: Code repositories, pull requests, issue tracking, CI/CD

### **Design & Collaboration**
- **Miro** - Visual collaboration and whiteboarding
  - URL: [miro.com](https://miro.com)
  - DRI: Product Team
  - Purpose: User journey mapping, wireframes, brainstorming sessions

### **Documentation & Knowledge Management**
- **Notion** - Documentation and project management
  - URL: [notion.so](https://notion.so)
  - DRI: All Teams
  - Purpose: Team documentation, meeting notes, project planning

### **Diagramming & Flowcharts**
- **Draw.io** - Technical diagrams and flowcharts
  - URL: [draw.io](https://draw.io)
  - DRI: Engineering Team
  - Purpose: System architecture diagrams, flowcharts, technical documentation

### **Communication**
- **Discord** - Team communication and chat
  - URL: [discord.com](https://discord.com)
  - DRI: All Teams
  - Purpose: Daily communication, quick discussions, voice calls

### **File Storage & Sharing**
- **Google Drive** - File storage and collaboration
  - URL: [drive.google.com](https://drive.google.com)
  - DRI: All Teams
  - Purpose: Document storage, file sharing, collaborative editing
